1、概述
	LEP 解释器是一个用于执行 LEP 脚本的控制台程序。LEP 解释器基于 C# 编写。
	LEP 解释器的调用方式如下："LEP Intepreter" [/m][/h][filename]

2、无参数模式
	若调用解释器时未传入参数，则解释器进入无参数模式。
	解释器在无参数模式下执行时，首先会弹出一个“打开”对话框，选择一个脚本文件打开即可进入执行模式执行脚本。
	若点击“取消”，则进入命令行模式：输入一行脚本，解释器将执行该行脚本。声明的变量、定义的函数将会保留。输入文件结束符（Ctrl + Z）以退出命令行模式。
	脚本执行结束时，会提示 "Finished."。请按回车键以退出解释器。

3、带参数模式
	I、概述
		LEP 解释器在调用时可以传入参数，但只能传入一个，多余的参数将会被忽略。此时解释器进入带参数模式。


	II、/m 参数及 /h 参数
		/m 及 /h 参数用于显示说明文档，/m 显示 LEP 语言说明文档（Readme_LEP.md），/h 显示解释器说明文档（Readme_Intepreter.md，即本文）。


	III、filename 参数
		filename 参数用于执行指定的脚本 filename。可以使用绝对路径或相对路径。解释器执行情况与无参数模式下的执行模式类似，但不会弹出“打开”对话框。
